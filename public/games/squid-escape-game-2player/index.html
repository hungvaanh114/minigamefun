Squid Escape Game 2Player

Hướng dẫn cách chơi / Instructions:

You must escape from this iron-covered labyrinth prison. Both blue players need to escape the prison. You must collect all the geometric shapes. Collect all the shapes and find both keys. Find the keys and run to the door. Be very careful, there are thorny obstacles on the ground. Dont stray too far from your friend; if you lose each other, you will lose the game.

Instructions

Use the WASD keys and the Arrow keys to move Both players must reach the door safely and complete the level You must collect all the shapes in the level
