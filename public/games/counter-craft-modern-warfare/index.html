Counter Craft: Modern Warfare

Hướng dẫn cách chơi / Instructions:

Counter Craft: Modern Warfare is a first person shooter which features modern warfare shooting in blocky world where your mission is to find the exit on each level and shoot all blocky creatures on our way using vast array of modern weapons.

Instructions

WASD - Move LMB - Shoot RMB - Aim Flashlight Space - Jump R - Reload G - Grenade C - Crouch Shift - Run Mouse Wheel - Change Weapon
