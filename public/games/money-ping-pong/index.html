Money Ping Pong

Hướng dẫn cách chơi / Instructions:

Money ping pong is a casual idle game where you need to deftly install money blocks, use them to reduce the space for balls and earn more game money. There are many different maps and dozens of functional blocks Relaxing gameplay

Instructions

Place money blocks on the field to create your own pinball strategy for making game money Buy upgrades to complete the level
