My First Robot

Hướng dẫn cách chơi / Instructions:

This is an educational game to assemble robots for kids. All the robots in the lab are incomplete,lets fix it together. The game has 6 unique robots with their gestures. It will be very interesting when we build it from scratch. You need to assemble the body parts into a complete machine and draw circuit components.
