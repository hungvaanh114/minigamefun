Hexa Stack Sort

Hướng dẫn cách chơi / Instructions:

Hexa Stack Sort is a fun and engaging puzzle game that will put your brain to the test! The goal is simple, but the challenge is exciting. You need to match and stack hexagonal blocks of the same color. You can move the blocks around. but be careful not to get stuck. Only blocks of the same color can be stacked together.
