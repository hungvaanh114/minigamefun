Cleaning Princess

Hướng dẫn cách chơi / Instructions:

This is an ASMR game for girls. If you like to clean everything, our game is right for you. The game has 5 modes: ear cleaning, heel cleaning, back spa, teeth cleaning, and fan cleaning. You can use different tools to clean and massage your body parts. But it is important to note that each mode has a countdown time and you must choose the right tool to do the cleaning.
