2048 Cubes

Hướng dẫn cách chơi / Instructions:

Endless 2048 game with a twist, blocks with physics. Blocks will colliide and bounce after collision. Drop and merge same numbers to reach higher numbers.

Instructions

Swipe right and left to move the block and release to drop Merge blocks with same number to get a bigger block Increase your score with more blocks Game ends when the board fills up to the dashed line
