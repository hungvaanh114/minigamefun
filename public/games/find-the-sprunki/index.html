Find the Sprunki

Hướng dẫn cách chơi / Instructions:

Find out the hidden Sprunki in the specified images. Each level has 12 hidden Sprunki. There are 12 levels in total. The time is limited so be fast and find all hidden objects before time runs out. Clicking in the wrong place several times reduces the time by an additional 5 seconds. So, if you are ready, start the game and have fun!
